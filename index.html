Ejercicio #1

Listado de todos los usuarios registrados en la base de datos.

  • Nombres
  • Apellidos
  • Correo
SELECT name, lastname, email FROM `users`;

Ejercicio #2

Listado de departamentos registrados en la base de datos.

  • Nombre del departamento
  • Nombre del pais
SELECT d.name AS 'Nombre del departamento', c.name AS 'Nombre del país' FROM departaments d JOIN countries c ON d.countries_id = c.id;

Ejercicio #3

Listado de escuelas por un país determinado.

  • Nombre de la escuela
  • Nombre del pais
SELECT s.name AS 'nombre de la escuela', c.name AS 'nombre del país' FROM schools s INNER JOIN countries c ON s.countries_id = c.id WHERE c.name = 'argentina';

Ejercicio #4

Listado de usuarios ordenados (de mayor edad a menor edad).

  • Nombres
  • Apellidos
  • Fecha de nacimiento
SELECT name AS 'nombres', lastname AS 'Apellidos', date_birth AS 'Fecha de nacimiento' FROM users ORDER BY date_birth ASC;

Ejercicio #5

En que departamento hay más usuarios registrados.

  • Departamento
  • Id
SELECT d.name AS 'Departamento', d.id FROM departaments d INNER JOIN users u ON d.id = u.departaments_id GROUP BY d.name, d.id ORDER BY COUNT(u.id) DESC LIMIT 1;

Ejercicio #6

Cual es el país con más usuarios registrados.

  • Pais
  • Id
SELECT c.name AS 'País', c.id FROM countries c JOIN departaments d ON c.id = d.countries_id GROUP BY c.name, c.id ORDER BY COUNT(d.id) DESC LIMIT 1;

Ejercicio #7

Listar solo 3 departamentos.

  • Nombre del departamento
  • Id
SELECT name AS 'nombre departamento', id FROM departaments LIMIT 3;

Ejercicio #8

Listar la información de un usuario determinado.

  • Nombres
  • Apellidos
  • Correo
  • Nombre del colegio
  • Nombre del departamento
  • Nombre del pais
SELECT u.name AS 'nombres', u.lastname AS 'apellidos', u.email AS 'correo', s.name AS 'nombre del colegio', d.name AS 'nombre del departamento', c.name AS 'nombre del país' FROM users u JOIN schools s ON u.schools_id = s.id JOIN departaments d ON u.departaments_id = d.id JOIN countries c ON d.countries_id = c.id WHERE u.id = '11';

Ejercicio #9

Listado de usuarios con un nombre determinado “and” .

  • Nombres
  • Apellidos
SELECT name AS 'Nombres', lastname AS 'Apellidos' FROM users WHERE name LIKE '%and%' OR lastname LIKE '%and%';

Ejercicio #10

Cuantos usuarios viven en un departamento determinado

  • Cantidad de usuarios “Numero”
SELECT COUNT(*) AS 'numero' FROM users WHERE departaments_id = 1;

Ejercicio #11

Listado de usuarios que tengan más de 20 años.

  • Nombres
  • Apellidos
  • Fecha de nacimiento
SELECT name AS 'Nombres', lastname AS 'Apellidos', date_birth AS 'Fecha de nacimiento' FROM users WHERE date_birth <= DATE_SUB(CURDATE(), INTERVAL 20 YEAR) ORDER BY ASC;

Ejercicio #12

Listado de usuarios que tengan entre 20 y 60 años.

  • Nombres
  • Apellidos
  • Nombre capital
SELECT u.name AS 'Nombres', u.lastname AS 'Apellidos', c.capital AS 'Nombre capital' FROM users u JOIN departaments d ON u.departaments_id = d.id JOIN countries c ON d.countries_id = c.id WHERE u.date_birth >= DATE_SUB(CURDATE(), INTERVAL 60 YEAR) AND u.date_birth <= DATE_SUB(CURDATE(), INTERVAL 20 YEAR);

Ejercicio #13

Listado de colegios que tengan el numero “7” en su número de teléfono

  • Nombres
  • Apellidos
  • Correo
  • Nombre del departamento
SELECT name AS 'Nombre colegio', phone AS 'Número de teléfono' FROM schools WHERE phone LIKE '%7%';

Ejercicio #14

Listado de usuarios que no estén asociados a ningún departamento

  • Nombres
  • Apellidos
  • Correo
  • Nombre del departamento
SELECT name AS 'Nombres', lastname AS 'Apellidos' FROM users WHERE departaments_id IS NULL;
